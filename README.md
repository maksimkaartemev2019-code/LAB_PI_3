# LAB_PI_3

WPF-приложение ToDo, выполненное по паттерну MVVM.

## Возможности

- просмотр задач списком;
- фильтрация: все, активные, выполненные;
- добавление задач с валидацией названия и описания;
- редактирование названия, описания и статуса;
- удаление одной задачи с подтверждением;
- удаление выбранных задач списком;
- сохранение задач между запусками в локальный JSON-файл;
- имитация асинхронных операций с индикатором загрузки.

## Архитектура

- `Models` - бизнес-данные и правила валидации.
- `Services` - локальное JSON-хранилище и подтверждение действий.
- `ViewModels` - состояние экрана, команды и логика отображения.
- `Views` - XAML-представления с привязками.
